## Description

This application is a online bookstore. Users can find and save their favorite books. The app is created with MERN stack, using React for the front end and MongoDB for the database.

heroku:

https://book-search-engine-janelu.herokuapp.com/

## Installation

- npm init
- React
- Express

* Apollo server express
* Graphql

## Usage

After installation, users can run npm start to connect
